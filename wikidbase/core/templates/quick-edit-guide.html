Quick wikidpage editing guide: wiki and database

wiki: writing page text

You can enter any text within a page, and that text can be structured (e.g. with heading levels, bullet points, bold and italic, etc.) using a simple open document format known as restructured text. Here's an example of the basics of restructured text: 
Main Title
==========

Here are some bullet points:

* Eggs
* Beans
* Spam

Sub Title 1
-----------

This is *italic* and this is **bold**.
For more information on using restructured text, see re-structured text formatting.

wiki: page linking

To link pages wiki-style, type wikiwords like this: 
MyPage
FunnyStory
or in brackets like this: 
[[My Report]]
[[Staff Page]]
These will automatically be recognised as links to wikidpages. If the named wikidpage does not exists, then after you have saved this page, you may click on the question mark to create it.

database functionality

Define wikidpage fields using this special markup:
  • Basic fields: 
    Name:: Fred
Date of birth:: 31/5/80
Can drive:: yes
  • Multiline fields: 
    Address::{
The street
The town
The city
}
  • Commands and queries to embed in a page (e.g for a particular report): 
    :::tasks:::
:::animals type = "Gold fish":::
:::show me staff and members where name contains "John":::
:::show me animals where name = Tigger and owner.name = Fred:::
:::show me anything where body contains "Hello" and modified < yesterday:::
